生词表2：职业相关

假名 声调 日语书写形式 释义
がく　せい 0 学生 学生
せん　せい 3 先生 先生
りゅう　がく　せい 3 留学生 留学生
きょう　じゅ 0 教授 教授
しゃ　いん 1 社員 普通职员
かい　しゃ　いん 3 会社員 上班族

补充

  • 学生がくせい

    • 广义：指学生这种身份
    • 狭义：专指大学生

  • 先生せんせい
    可以前加名字，称呼老师，医生作家，画家，律师等

  • 社員しゃいん
    日语的社員类似于员工的意思，一般前接公司名，仅仅表示属于某公司的人

  • 会社員かいしゃいん
    日语的会社員表示一种职业，类似医生，工程师，意思是公司职员，在公司工作的人

生词表4：人称指代

假名 声调 日语书写形式 释义
わたし 0 私 わたし 我
あなた 2 あなた 你
ちち 0 父 (我)父亲
おう 1 王 中国人姓氏
ちょう 1 張 中国人姓氏
もり 0 森 日本人姓氏
はやし 0 林 日本人姓氏
おの 0 小野 日本人姓氏

补充

  • 私わたし

    • わたし：第一人称我，男女都可以用，比较正式，对话中经常忽略
    • 僕　ぼく：男性的自称，用于熟人间
    • 俺　おれ：男性的自称，用于和平辈，或比自己级别低的人说话时

  • あなた

    • あなた：多用于妻子对自己丈夫的称呼。或用于不知道对方的姓名又必须打招呼的时候
    • 君　きみ：你，对男女都可以用，用于上对下，比自己年龄小的人之间
    • 俺 おれ：男性的自称，用于和平辈，或比自己级别低的人说话时
    • お前　おまえ：一种不礼貌的说法，上对下才用

  • 父ちち

    • ちち：用作和别人谈到自己父亲的时候，是自谦的说法
    • お父さん：称呼别人父亲的时候，或家里叫自己父亲的时候，表示尊敬的说法
