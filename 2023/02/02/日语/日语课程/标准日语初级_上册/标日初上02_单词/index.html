标日初上02_单词

生词表1：书本文具

假名 声调 日语书写形式 释义
ほん 1 本 书
ノート 1 ノート 笔记本
て　ちょう 0 手帳 记事本
えん　ぴつ 0 鉛筆 铅笔
しん　ぶん 0 新聞 报纸
ざっ　し 0 雑誌 杂志
じ　しょ 1 辞書 词典

补充

  • 手帳
    随身携带的、按照日期或时间记录事件、备忘的小册子。强调时间性和便捷性
  • ノート
    为了避免忘记而做的记录，通常比较有系统性，比如上课笔记
  • 中文中的新闻在日语中使用”ニュース”(news)

生词表7：其他

假名 声调 日语书写形式 释义
かい　しゃ 0 会社 公司
かた 2 方 (敬称)位、人
ひと 0 人 人
か　ぞく 1 家族 家人、家属
はは 1 母 (我)母亲
おがあさん 2 お母さん 母亲
に　ほん　ご 0 日本語 日语
ちゅう　ごく　ご 0 中国語 中文

补充

  • 母亲
    • はは: 和别人提到自己的母亲时用的自谦语
    • お母さん: 对他人的母亲或是在家里称呼自己母亲的时候用的尊敬语

亲属的称谓

释义 自己的亲属 别人的亲属
祖父、外祖父 そふ　祖父 おじいさん　お祖父さん
祖母、外祖母 そぼ　祖母 おばあさん　お祖母さん
父母 りょうしん　両親 ごりょうしん　ご両親
父亲 ちち　父 おとうさん お父さん
母亲 はは　母 おがあさん お母さん
儿子 むすこ　息子 むすこさん　息子さん
女儿 むすめ　娘 むすめさん　娘さん
- - じょうさん　お嬢さんさん
